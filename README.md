## Building from source

This book is written in [Typst](https://typst.app/docs/tutorial/). To compile it
to a PDF, install the [Typst compiler](https://typst.app/open-source/#download),
and run

```shell
typst compile starry-plough-dances.typ
```

Optional: to reduce the PDF file size by 65%, install
[pdfcpu](https://pdfcpu.io/getting_started/install) and run

```shell
pdfcpu optimize starry-plough-dances.pdf
```
